Instalação
=====
## Pre-requisitos

* Node - http://nodejs.org/
* Git - http://git-scm.com/

´Depois de instalar o node, use `npm` para installar o Grunt.

    npm install -g grunt-cli
    
## Quick Start

Entre na pasta assets, instale as dependencias e execute `grunt`:

    cd assets
    npm install
    grunt
